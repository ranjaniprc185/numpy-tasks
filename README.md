# numpy-tasks
This repository contains multiple NumPy exercises used for Python Data Analytics practice.

Skills:
- Array manipulation
- Diagonal operations
  
Task 1: Matrix Below Diagonal
Create a 5x5 matrix with values 1, 2, 3, 4 just below the diagonal.
  
Task 2: Common Values Between Two Arrays
Find common values between two NumPy arrays.

Task 3: Dates Corresponding to July 2016
Generate all dates for the month of July 2016.

Task 4: Subtract Mean of Each Row
Subtract mean of each row from the matrix.

Task 5: Diagonal of Dot Product
Find the diagonal elements of a dot product.

Task 6: Bootstrap 95% Confidence Interval
Compute bootstrapped 95% confidence intervals for the mean of a 1D array.

All output screenshots will be stored inside the `/outputs/` folder
